Ctrl.Framework
==============

![LICENSE](https://img.shields.io/github/license/ctrlcommunity/Ctrl.Framework?style=plastic)

前言
=====

ASP.NET Core3.1、AbpvNext、BootStrap进行构建的通用型基础开发框架.

项目介绍
=====
**Ctrl.Framework**是一个升级版基础权限开发型框架，支持权限管理，代码生成，日志审计等。
